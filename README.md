## 🚀 试试

1. **本地调试**

   安装依赖：

   ```shell
   yarn
   ```

   本地运行：

   ```shell
   yarn develop
   ```

   然后你就可以访问 `http://localhost:8000` 了

   也可以访问 `http://localhost:8000/___graphql`来试试 GraphQL
